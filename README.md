# CANLAH.AI Website

Company website for CANLAH AI — product matrix and platform overview.

## What This Does

The official CANLAH AI website showcasing the company's product suite, team, client testimonials, platform capabilities, pricing, and FAQ. Built with a dark SaaS design system using navy, cyan, and blue accent colors. Supports internationalization.

## Tech Stack

- Nuxt 4 (Vue 3, SSR)
- Tailwind CSS
- i18n (multilingual support)
- Vercel (auto-deploy)

## Setup

```bash
git clone https://github.com/Canlah-AI/canlah-website.git
cd canlah-website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Sections

Header, Hero, Stats, How It Works, Testimonials, Comparison, Products, Platform, Studio, Pricing, FAQ, CTA, Footer.
