# CLAUDE.md

## What This Is
Company website for Canlah AI (canlah.ai). Dark SaaS design, bilingual (EN/ZH).

## Tech Stack
- Nuxt 4 + Tailwind CSS
- Deployed on Vercel (auto-deploy on push)

## How to Run
```bash
npm install
npm run dev
```

## No Database
Static marketing site. No Supabase needed.

## Brand Rules
- Navy #050d1a, Cyan #00d4ff, Blue #0d47e0, Green #00e5a0
- Dark theme only
